1. オペレーティングシステムへのインストール(Windows 95,98,Me,XP,Vista,7)
-------------------------------------------------------------------------------
    Windows環境の場合はダブルクリックしてフォントビューアを開き、システムにイン
    ストールするか、直接(ルートディレクトリ)\Windows\fontにドラッグしてコピーし
    ます。この操作には管理者権限を必要とします。
    
    なお、管理者権限がない場合にはAMP Font Viewerというフリーソフトウェアを用い
    ると一時インストールという形でフォントを利用することができます。(このソフト
    ウェアのインストール自体も管理者権限は不要です)
    
    このソフトウェアは日本語、英語に対応しています。
	

1. To install this font in operating system(Windows 95,98,Me,XP,Vista,7)
-------------------------------------------------------------------------------
    When you use Windows , please double-click this font and open font viewer 
    or drag font file for (root-directory)\Windows\fonts to install font.
    This control needs admin right.
    
    If you don't have admin right , you can temporary install this font by 
    Free softwear "AMP Font Viewer". (You can install "AMP Font Viewer" 
    without admin right.)
    
    It is avaiable Japanese and English.
